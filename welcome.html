Seeing this page means that you have successfully installed Lively Kernel on your computer!
This Lively installation allows you to run and modify the Lively Kernel core system which is hosted on Github on your own computer. Additionally to the core code, the Webwerkstatt's PartsBin is available by default.
If you would like to contribute to the core development please consider forking the core repository and submitting a pull request. (If you want to learn more about working with Github please see github-fork-a-repo)
New to Lively?
To start working on the core system you can create your own persistent "workspaces". Simply save this or the blank world under a new name (World menu -> Save world as...). To view and edit the source code use the System code browser.
