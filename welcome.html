How to work with lively-web.org?
Before you start please login. Your username is mapped to your user directory on lively-web.org. In there you can create new worlds and modules. If you have not created any content yet please click here to create a new workspace in your user directory.
If you want to change or add new content, just visit a world and save the world under a different name (World menu -> Save world as...).
